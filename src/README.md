# AR Inspector - YOLOv8

Projeto desenvolvido em Python utilizando YOLOv8 para deteccao de objetos em tempo real por meio da webcam.

## Funcionalidades

- Deteccao de objetos em tempo real
- Interface com HUD
- Modos INFO, SIMPLES e DEBUG
- Ajuste de confianca durante a execucao
- Captura de screenshots
- Compatibilidade com YOLOv8 (Ultralytics)

## Tecnologias

- Python 3
- OpenCV
- Ultralytics YOLOv8
- NumPy

## Instalacao

```bash
pip install ultralytics opencv-contrib-python numpy
```

## Execucao

```bash
python ar_yolo_corrigido_ascii.py
```

## Controles

| Tecla | Funcao |
|-------|---------|
| M | Alterna modo |
| P | Pausa |
| H | Ajuda |
| C | Screenshot |
| + / - | Ajusta confianca |
| ESC | Sair |
